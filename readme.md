# KeenKeeper

KeenKeeper is a modern relationship management web application designed to help you stay connected with the people who matter most.  
It allows you to track interactions, manage friendships, and maintain meaningful relationships in an organized way.

---

## Features

- Friend management system
- Interaction tracking
- Timeline view of interactions
- Analytics dashboard with charts
- Status indicators
- Fully responsive design

---

## Tech Stack

- React
- React Router DOM
- Tailwind CSS
- Lucide React
- Recharts
- LocalStorage
